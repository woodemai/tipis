# Анализ гармонических и цифровых сигналов

Этот проект генерирует гармонические и цифровые сигналы для различных частот, вычисляет их спектры с помощью преобразования Фурье (FFT) и отображает результаты на графиках. Программа позволяет визуализировать сигналы во временной области и их амплитудные спектры.

## Описание

Программа создает два типа сигналов:
- **Гармонические сигналы** (синусоидальные функции с амплитудой 2).
- **Цифровые сигналы** (квадратные волны, полученные через знак синуса).

Для каждого сигнала программа:
1. Строит графики временной области для сигналов с разными частотами.
2. Строит амплитудные спектры этих сигналов, используя преобразование Фурье (FFT).
3. Отображает результаты в виде графиков.

Программа генерирует сигналы для частот 1 Гц, 2 Гц, 4 Гц и 8 Гц.

## Требования

Для запуска программы на вашем компьютере должны быть установлены следующие библиотеки Python:
- `numpy` — для численных вычислений и преобразования Фурье.
- `matplotlib` — для визуализации сигналов и спектров.

Можно установить их с помощью `pip`:

```bash
pip install numpy matplotlib
```

## Как запустить

1. Скачайте или клонируйте репозиторий на ваш компьютер.
2. Запустите программу:

```bash
python main.py
```

Программа отобразит графики для каждого типа сигнала (гармонического и цифрового) и их спектров для каждой из заданных частот.

## Описание кода

- `generate_sine_signals(frequencies, amplitude, time, initial_phase)` — генерирует гармонические сигналы для заданных частот.
- `generate_square_signals(time, frequencies, amplitude)` — генерирует цифровые сигналы для заданных частот.
- `calculate_spectrum(signals)` — вычисляет амплитудные спектры для каждого сигнала с использованием FFT.
- `configure_subplot(ax, x_data, y_data, title, xlabel, ylabel, color, xlim=None)` — вспомогательная функция для настройки подграфиков.
- `plot_signals_and_spectra()` — основная функция, которая генерирует и отображает все графики для сигналов и их спектров.

## Пример вывода

Программа отображает 4 ряда графиков:
1. Гармонический сигнал для каждой частоты (1 Гц, 2 Гц, 4 Гц, 8 Гц).
2. Цифровой сигнал для каждой частоты.
3. Амплитудный спектр гармонического сигнала.
4. Амплитудный спектр цифрового сигнала.
